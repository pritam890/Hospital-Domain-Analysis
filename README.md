# 🏥 Emergency Room Data Analysis Project (Excel Dashboard)

This project presents a **complete end-to-end Emergency Room (ER) Data Analysis** solution using **Microsoft Excel**, tailored for Data Analysts, Excel Enthusiasts, and Healthcare Professionals aiming to enhance **data-driven decision-making** within hospital emergency departments.

## 📌 Project Overview

In this project, we developed an **interactive dashboard** that delivers critical insights into emergency room performance using **Power Query, Power Pivot, DAX**, and **data visualization techniques**. It provides hospitals with actionable metrics to help streamline operations, identify bottlenecks, and improve patient experience.

## 🔍 Key Insights Delivered

The dashboard offers the following analytics capabilities:

* ✅ **Total Number of Patients** visited the ER
* ✅ **Average Wait Time** before seeing a patient
* ✅ **Patient Admission Status** (Admitted vs Not Admitted)
* ✅ **Age Distribution** of patients
* ✅ **Gender-Based Analysis**
* ✅ **Department Referrals** for various cases
* ✅ **Percentage of Patients Seen Within 30 Minutes**

## 🛠️ Tools & Techniques Used

* **Power Query**: For data extraction and transformation
* **Power Pivot**: To define relationships and create DAX measures
* **Pivot Tables**: For dynamic report generation
* **Excel Charts & Slicers**: To create an interactive and user-friendly dashboard

## 📊 Benefits

* Enable hospital administrators to monitor ER efficiency
* Identify patient demographics and trends
* Measure timely service performance
* Guide policy decisions based on data evidence
